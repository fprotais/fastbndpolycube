# fastbndpolycube

# Use CMake to build the project:
```sh
git clone --recurse-submodules https://github.com/fprotais/fastbndpolycube &&
cd fastbndpolycube &&
mkdir build &&
cd build &&
cmake .. &&
make -j 
```

# Running the code :

```sh
./fastpolycube ../boundary.obj ../labeling.txt result.obj
```
